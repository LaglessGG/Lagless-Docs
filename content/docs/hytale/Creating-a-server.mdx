---
title: Creating a Hytale Server
description: How to authorize and start your Hytale server.
---

# Game Selection

- Go to the *Game Selection* section.
- Choose **Hytale**.

# Server Authorization

- Click **Start**.
- A popup will appear asking you to authorize the server.

## Device Verification

- Click the link in the popup, or open it manually:
  https://oauth.accounts.hytale.com/oauth2/device/verify
- Enter the provided code to authenticate.

<Callout type="info" title="Authentication Required">
Authorization is required both for downloading and starting the server, so this step must be completed twice.
</Callout>

# Starting the Server

- After completing authorization, start the server again.
- The server will now finish downloading and launch normally.
