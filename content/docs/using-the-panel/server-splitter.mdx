# What is a server splitter?

A server splitter is a way to split your resources into "sub-servers" to run other various processes such as discord bots, backend servers, or proxies.

<Callout type="info" title="Note">
  If you choose to upgrade your server specs, additional resources are allocated to your primary server. You can later determine if you need to increase or decrease resources to your other sub-servers.
</Callout>

# Creating Your Splits

Clicking on your primary server, go to <b>Utilities \> Splitter</b>. Here you can see how many splits your server has or is using.

![splitter-create](/images/splitter-create.png)

Click the Plus to create a split and fill out the necessary information.

If you want your sub-users from your main server to access the split, make sure to tick <b>Sync Subusers</b>.

# Updating a Server Split

![splitter-update](/images/splitter-update.png)

Should you need to remove a split to free up resources, you can click on the trashcan icon to delete it.
You can always increase or decrease a certain resource, however make sure you select <b>Update Server</b> after making changes, so it will take effect.

Clicking on the <b>Open</b> button, you can go to your sub-server to install or manipulate the resources as you need.
