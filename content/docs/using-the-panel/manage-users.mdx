# Create a Subuser

Do you have a service with us and do you want to create a new subuser?  
This is how you do it. Make sure you are logged into our game panel.

1. Go to the "Users" tab on the left. (as shown in the image below).
2. Click on "New user".

![Create user](/images/panel/users/create_user.png)


3. Enter the email address of the person you want to invite.
4. You can tick the selection box for all permissions in a section.
5. Or you can select each individual permission.
6. Click invite user

![New user permissions](/images/panel/users/New_user_permissions.png.png)

If the user is new and they don't have an account yet, they will receive an email about that.

![New account created](/images/panel/users/email_account_creation.png)

When they have an account, they will be added as a subuser.

![Added subuser](/images/panel/users/added_subuser.png)

# Modify a Subuser's Permissions

If you wish to modify the permissions of a subuser

1. Click on the users tab.
2. Modify permissions

![Modify permissions](/images/panel/users/modify_permissions.png)
