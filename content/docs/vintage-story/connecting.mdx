## Introduction

Once you have finished configuring your Vintage Story server, you can start your server for players to connect and play. Vintage Story allows players to connect via direct connection or via the in-game server list.

## Obtaining Server Information

You'll find the required information from your Vintage Story server's control panel to join your server.

- **IP Address**: Can be found on the home page of the control panel  
  ![Server IP Address](/images/vintagestory/serverip.png)
- **Server Name**: Configured in your `serverconfig.json` (see [Server Setup Guide](/vintage-story/server-setup#initial-server-configuration))

## Direct Connect

You can connect to your server using direct connect by adding the server to your favorites.

<Steps>
    <Step>
    Launch Vintage Story and click **Multiplayer** from the main menu
    </Step>
    <Step>
    Click **Add New Server** to create a new server entry
    </Step>
    <Step>
    Enter your server's IP address and port (format: `IP:Port`)
    </Step>
    <Step>
    Optionally, give it a custom name for easy identification  
    *Note: This is for your reference only and doesn't need to match the server's actual name*
    </Step>
    <Step>
    Click **Add Server** to save the entry
    </Step>
    <Step>
    Select your server from the list and click **Connect**
    </Step>
</Steps>

## Browsing Server List

You can use your server name to search the server list for your Vintage Story server.

<Steps>
    <Step>
    Launch Vintage Story and click **Multiplayer** from the main menu
    </Step>
    <Step>
    Click **Browse Public Servers** to open the server browser
    </Step>
    <Step>
    Use the search bar to enter your server's name
    </Step>
    <Step>
    Locate your server in the results list
    </Step>
    <Step>
    Click your server and then click **Connect**
    </Step>
</Steps>
