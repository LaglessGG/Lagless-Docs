## Converting your icon

Select the server icon you want to use and upload it in [MCUtils](https://mcutils.com/server-icon-converter).

![mcutils](/images/minecraft/mcutils.png)

Next, go to your server and open the Files Manager. From here, you can 
select on <b>Change Icon</b> then select the downloaded image.

![change-icon](/images/minecraft/change-icon.png)

Start your server and your icon should appear..

## Troubleshooting 

If you are having issues with <b>Change Icon</b>, you can upload it directly to your File Manager.

![upload-icon](/images/minecraft/upload-icon.png)

Once the file is uploaded, make sure the file is named `server-icon.png`. Start your server and verify the image is showing up.
