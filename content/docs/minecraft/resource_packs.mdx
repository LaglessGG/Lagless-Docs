## Introduction

This guide uses the **Bare Bones** resource pack for a Minecraft 1.21.8 server as an example. You can download this **resource pack** [here](https://modrinth.com/resourcepack/bare-bones/versions?g=1.21.8)

<Steps>
<Step>
Find or create a resource pack you would like on your server.
</Step>
<Step>
Upload this resource pack to [mc-packs.net](https://mc-packs.net/)

![mc-packs](/images/minecraft/resource-packs/mc-packs.png)
</Step>
<Step>
When uploaded, you'll receive two links. Copy both
![mc-packs](/images/minecraft/resource-packs/resource-pack-links.png)
</Step>
<Step>
Paste both links into your `server.properties` file, all the way at the bottom
![mc-packs](/images/minecraft/resource-packs/server.properties.png)
</Step>
<Step>
now delete the 2 lines you just pasted in, `resource-pack=` & `resource-pack-sha1=`
</Step>
<Step>
Save the file
</Step>
<Step>
Connect to your server and download the files when asked.
![mc-packs](/images/minecraft/resource-packs/connect-and-download.png)
</Step>
<Step>
If you made it this far, congratulations, you now have an awesome server resource pack.
</Step>
</Steps>
